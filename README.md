# Snackgnito 🍪🕵️‍♂️

**Snackgnito** (formerly Snackgnito) is a lightweight, privacy-focused Chrome Extension that allows you to open single-use, isolated tabs. It intercepts and manages cookies in a virtual container, ensuring they never reach your native browser storage.

This allows you to log into multiple accounts on the same website simultaneously without mixing up sessions or polluting your main browser's cookies.

## Features

- **Instant Isolation:** Open any link in an isolated Snackgnito tab from the context menu, or convert your current tab instantly with a keyboard shortcut (`Alt+V`).
- **Complete Session Separation:** Utilizes Chrome's Manifest V3 `declarativeNetRequest` API and injected proxy scripts to intercept `Set-Cookie` and `document.cookie` operations. 
- **Ephemeral by Default:** Once you close a Snackgnito tab, its virtual cookie jar is completely wiped from memory. No trackers, no leftover session data.
- **Child Tab Tracking:** Middle-clicking or opening a link in a new tab from an existing Snackgnito tab maintains the isolation automatically.

## How it Works

Snackgnito works by dynamically rewriting HTTP headers and proxying the JavaScript environment:
1. **Network Level:** Intercepts incoming `Set-Cookie` headers and strips them before the browser native cookie store can save them. It dynamically adds `Cookie` headers to outgoing requests using `declarativeNetRequest` session rules.
2. **DOM Level:** Injects a script into the `MAIN` execution world to proxy `document.cookie`, `localStorage`, and `sessionStorage`, tricking the website into thinking it has full access to browser storage while it's actually interacting with an isolated, in-memory Map.

### Manual Installation (Developer Mode)

1. Clone this repository:
   ```bash
   git clone https://github.com/nowshadabir/snagnito.git
   cd snackgnito
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the extension:
   ```bash
   npm run build
   ```
4. Load into Chrome:
   - Go to `chrome://extensions/`
   - Enable **Developer mode** in the top right corner.
   - Click **Load unpacked**.
   - Select the `dist/` directory generated by the build step.

## Development

Snackgnito is built with **TypeScript**, **Vite**, and **Tailwind CSS**.

```bash
# Run the development server with Hot Module Replacement (HMR)
npm run dev

# Build for production
npm run build
```
